# Requirements Baseline Index

This folder is the canonical V1 requirements baseline for Machine Shop Planning Software.

Use these documents for implementation:

- `PRD.md`: product intent, scope, product decisions, success criteria, release criteria
- `TECHNICAL_REQUIREMENTS.md`: architecture, stack, modules, APIs, runtime, testing
- `DATA_MODEL_REQUIREMENTS.md`: SQLite schema, table relationships, indexes, Excel schemas
- `BUSINESS_LOGIC_FORMULAS.md`: deterministic formulas, planning rules, recommendation logic
- `USER_EXPERIENCE.md`: user journeys, screen behavior, UX states, planner workflow

Do not implement from older drafts or archived background notes. The original source/background note is archived at `docs/archive/requirements_legacy.md`.

If a requirement changes, update the owning document above instead of duplicating logic in another file.
